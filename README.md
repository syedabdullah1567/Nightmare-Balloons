# Nightmare-Balloons

Nightmare Balloons is a 2D balloon popping game developed as a third semester pair project for Computer Organization and Assembly Language.

## Gameplay preview (Click on image)

[![Gameplay Video](mq4.jpg)](https://youtu.be/jR9vPILLJYU)

 # Tech Stack
 
- Assembly Language 8088

# Gameplay

- Match keyboard input with letter in balloon to pop a balloon
- Each balloon pop awards 10 points
- Balloon hitting the ceiling causes a reduction in lives
- Three lives in total
- Total gameplay time - 1 Minute
- Game automatically switches to a faster speed after 30 seconds
- Keyboard-based controls


# To run game:

Copy all files out of subfolders into a main folder and place the new folder in the compiler/IDE directory
